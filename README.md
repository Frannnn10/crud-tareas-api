# CRUD de Tareas 

Este proyecto es una aplicación para crear y administrar tareas.

Permite:

- Registrarse
- Iniciar sesión
- Crear tareas
- Ver tus tareas
- Editar tareas
- Eliminar tareas

Cada usuario solo puede ver y modificar sus propias tareas.

Las contraseñas están protegidas (no se guardan como texto normal) y no se puede acceder a las tareas sin iniciar sesión.

## Cómo usarlo

1. Instalar dependencias:
npm install

2. Ejecutar el servidor:
node server.js

3. Abrir el frontend con Live Server en VS Code.
